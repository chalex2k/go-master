# О языке Go

## Происхождение
- Создан в Google в 2007 году (Роберт Гриземер, Роб Пайк, Кен Томпсон)
- Анонсирован в ноябре 2009 года
- Open-source проект
- Актуальная версия: 1.26

---

# Философия Go

## Радикальная простота
- Всего 25 ключевых слов
- Нет наследования, исключений, перегрузки операторов
- Нет конструкторов/деструкторов
- Нет неявных преобразований типов
- Гарантия обратной совместимости

> Как заметил Роб Пайк: "Сложность мультипликативна — попытка решить проблему усложнением одной части неизбежно порождает сложность в других"

---

# Ключевые особенности

- **Быстрая компиляция** — статическая компиляция
- **Строгая типизация** — ошибки на этапе компиляции
- **Единый стиль** — `gofmt` форматирует код
- **Богатая стандартная библиотека**

---

# Параллелизм как фундамент

Go создавался для эпохи многоядерных процессоров:

- **Горутины** — легковесные потоки (тысячи и миллионы одновременно)
- **Каналы** — безопасное взаимодействие между горутинами
- **Модель CSP** — Communicating Sequential Processes

```go
go func() {
    ch <- data  // отправка в канал
}()
value := <-ch  // получение из канала
```

# Нулевые значения

В Go нет неинициализированных переменных:

| Тип | Нулевое значение |
|-----|------------------|
| Числа | `0` |
| bool | `false` |
| string | `""` |
| Указатели, срезы, map, каналы | `nil` |

---

# Числовые типы

**Знаковые:** `int8`, `int16`, `int32`, `int64`, `int`
**Беззнаковые:** `uint8`, `uint16`, `uint32`, `uint64`, `uint`
**Специальные:** `byte` (= uint8), `rune` (= int32)
**С плавающей точкой:** `float32`, `float64`

> `int` и `uint` — платформозависимые (32 или 64 бита)

---

```go
var x int = 10
var y float64 = 30.2

// sum := x + y  // ОШИБКА!

var sum float64 = float64(x) + y  // Правильно
```

# Строки и руны

**Строки — это последовательность байтов**

```go
s := "Hello, 世界"
fmt.Println(len(s))    // 13 байт (не 9 символов!)

r := '€'               // руна (int32)
fmt.Println(r)         // 8364

for i, r := range "Hello, 世界" {
    fmt.Printf("%d: %c\n", i, r)
}
```

---

# Подстроки и поиск

**Взять N символов (не байтов!):**

```go
s := "Hello, 世界"

// Неправильно: s[:5] может разрезать руну!
// Правильно: через конвертацию в руны
runes := []rune(s)
substr := string(runes[:5])  // "Hello"

// Количество рун
import "unicode/utf8"
n := utf8.RuneCountInString(s)  // 9
```

**Поиск подстроки:**

```go
import "strings"

idx := strings.Index(s, "世界")    // 7 — позиция в байтах!
idx := strings.Index(s, "Hello")  // 0

// Проверка наличия
if strings.Contains(s, "世界") {
    fmt.Println("найдено")
}
```

# Конвертация строк и чисел

```go
import "strconv"

// Строка → число
n, err := strconv.Atoi("123")         // 123, error
n, err := strconv.ParseInt("FF", 16, 64)  // 255 (16-ричное)
f, err := strconv.ParseFloat("3.14", 64)  // 3.14

// Число → строка
s := strconv.Itoa(123)                // "123"
s := strconv.FormatInt(255, 16)       // "ff" (16-ричное)
s := strconv.FormatFloat(3.14, 'f', 2, 64)  // "3.14"

// Ошибка конвертации
n, err := strconv.Atoi("abc")  // n=0, err!=nil
if err != nil {
    fmt.Println("не число!")
}
```

# Цикл for — единственный в Go

```go
// Полная форма
for i := 0; i < 10; i++ {
    fmt.Println(i)
}

// Как while
i := 1
for i < 100 {
    i = i * 2
}

// Бесконечный
for {
    // ...
    if done { break }
}
```

---

# for-range

```go
// Срезы
for i, v := range []int{2, 4, 6} {
    fmt.Println(i, v)
}

// Map (порядок случайный!)
for k, v := range map[string]int{"a": 1} {
    fmt.Println(k, v)
}

// Строка (по рунам!)
for i, r := range "Hello, 世界" {
    fmt.Printf("%d: %c\n", i, r)
}
```

---

# break и continue

```go
for i := 0; i < 10; i++ {
    if i == 5 { break }      // выход из цикла
    if i%2 == 0 { continue } // следующая итерация
    fmt.Println(i)
}
```

---

# Метки для вложенных циклов

```go
outer:
for _, sample := range samples {
    for i, r := range sample {
        if r == 'l' {
            continue outer    // к следующей итерации внешнего цикла
        }
        fmt.Println(i, r)
    }
}
```

# Оператор switch

```go
switch size := len(word); size {
case 1, 2, 3, 4:
    fmt.Println("short")
case 5:
    fmt.Println("medium")
default:
    fmt.Println("long")
}
```

**Нет fallthrough по умолчанию!**

---

# Пустой switch

```go
switch {
case score >= 90:
    fmt.Println("A")
case score >= 80:
    fmt.Println("B")
case score >= 70:
    fmt.Println("C")
default:
    fmt.Println("F")
}
```

# Затенение переменных

```go
x := 10
if x > 5 {
    x := 5    // НОВАЯ переменная! Затенение!
    fmt.Println(x)  // 5
}
fmt.Println(x)  // 10 (внешняя)
```

**Внимание:** `:=` может случайно создать затенение!
